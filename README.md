mvimrc
======

This is the repository of personal vim settings.

  * Installation:
  
    git clone xxx

    git submodule init
    
    git submodule update

  * Update plugin:

    cd /path/to/plugin
    
    git pull origin master

  * Update all plugins:

    git submodule foreach git pull origin master

  * Delele plugin:

    git rm bundle/plugin-name
